# Descripción

AutoLoginWT es una aplicación para guardar tu usuario y contraseña del juego, para luego iniciar sesión automáticamente, a través de una combinación de teclas para mayor facilidad (Ctrl+F10) o de los respectivos botones dentro de la aplicación.

Puedes descargar la aplicación desde la siguiente URL `(Versión Independiente Recomendado):`

> https://drive.google.com/drive/folders/17IcXGKn1XWnwuuBeldfgoidoKyBifMYv?usp=sharing

# Funcionamiento General

- Ejecute el archivo AutoLoginWT.exe Como Administrador.

[![Readme Card](https://res.cloudinary.com/dxblq473l/image/upload/v1674862379/1_hbcdgb.png)](https://res.cloudinary.com/dxblq473l/image/upload/v1674862379/1_hbcdgb.png)

- Selecciona "NyxLauncher.exe" en la ruta de instalación de WolfTeam.

[![Readme Card](https://res.cloudinary.com/dxblq473l/image/upload/v1674862781/2_zpyu5l.png)](https://res.cloudinary.com/dxblq473l/image/upload/v1674862781/2_zpyu5l.png)

>**Note**
>Esto solo es necesario, la primera vez que ejecutas la aplicación.

- Ingresamos en el primer recuadro, el tiempo (En Segundos) en el que se va escribir tu usuario y contraseña, una vez se ejecute el NyxLauncher. Luego escribimos 
nuestro Usuario y Contraseña en los respectivos recuadros. La aplicación guarda automáticamente los datos y los encripta para que no puedan ser usados por terceros.

[![Readme Card](https://res.cloudinary.com/dxblq473l/image/upload/v1674862904/3_pekx3k.png)](https://res.cloudinary.com/dxblq473l/image/upload/v1674862904/3_pekx3k.png)

>**Note**
>Si El "NyxLauncher.exe" no carga en el tiempo especificado puede que no funciona correctamente, `Si haces directos no captures el escritorio`,
>ya que se podría mostrar tu contraseña.

- Una vez que termines de ingresar tus datos, ya puedes hacer clic en el botón ReLogin para ingresar al juego.

[![Readme Card](https://res.cloudinary.com/dxblq473l/image/upload/v1674863124/4_buctzt.png)](https://res.cloudinary.com/dxblq473l/image/upload/v1674863124/4_buctzt.png)

- Esto nos permite hacer Relogin al presionar (Ctrl + F10) muy útil cuando se congela el juego.
